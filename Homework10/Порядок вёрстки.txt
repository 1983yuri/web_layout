1. создаем html, css, img
2. подключаем css в html, если верстаем на bootstrap не забываем помимо его подключения в шапке, подключить <script> внизу, ближе к </body>
3. накидываем весь контент из макета в html, предварительно, продумав все блоки
4. поключаем шрифты, лучше локально
5. позиционируем(flex, grid, position: relative, position: absolute, задаем padding и margin, создаем блоки центровки, через: margin 0 auto-width:100%, либо через padding-left, padding-right(calc) )
6. делаем медиазапросы в отдельном css и не забываем его подключить в html(адаптив)
7. тестируем и проверем в отладчике для всех разрешений и устройств (увидеть устройства ctrl + shift + m)
8. создаем анимацию используя в отладчике: hov
9. делаем теги HTML5
10. прогонаем css через автопрефиксер: https://autoprefixer.github.io/ru/
11. прогоняем html через валидатор и исправляем ошибки: https://validator.w3.org/
12. Проект готов, отдаем заказчику
